Interfacing LED Display

Part I

In this part of the exercise we will interface a 7-segment LED display to port 0.The schematic for the hardware is given below. All resistors are about 360 ohms. Be sure to pay attention to details such as the wire colors and orientation of the plug-in connectors.

Write a program to write random HEX numbers to the LED whenever the pushbutton switch is pressed.

  • When the pushbutton is pressed, read the value of a running timer to obtain the random value. (You will need to turn one of the timers on)
  • Convert the least significant nibble of the value to the corresponding 7-segment value. (A subroutine should be used for this). Note that the LED segments are ACTIVE LOW because a common anode display is used.
  • Send the converted value to port 0. (Don't forget to configure port 0 as an output, and enable the crossbar switch).

Run the program to the beginning of the subroutine (over and over) and ensure that the value you are reading from the timer is the value that is being displayed.

Part II

In the part of the exercise we are going to combine the temperature sensor program and a modified version of your display code. The final program should continuously display the least significant nibble of the most significant byte of the temperature code (read from ADC0). Note that the ADC0 is set for left justified data.
